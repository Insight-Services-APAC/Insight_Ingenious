# Installation

Follow these steps to set up the **Ingenious** package locally:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Insight-Services-APAC/Insight_Ingenious.git
    ```

2. **pip Install**:
    ```bash
    pip install dist/ingenious-1.0.0-py3-none-any.whl
    ```
